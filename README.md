# Projects
Using mathematical areas of Probability, Linear Algebra and Calculus, I have created these projects in Python using predominantly the NumPy library. These Stochastic models seek to model financial markets and risk, mainly delving into financial risk in markets and credit objects. Each model is accompanied by supporting comments to guide logic.
